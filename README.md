# MRGPL - Arzooo Email Template

This repository contains the HTML email template for Arzooo's promotional emailer featuring USHA products.

## 📁 Files

- **arzooo_usha_emailer_v3_A.html** - Main email template with embedded CSS and responsive design

## ✨ Features

- 📱 **Fully Responsive** - Optimized for mobile (≤380px), tablet (≤540px), and desktop
- 🎨 **CSS Design Tokens** - Easy customization with CSS custom properties
- ♿ **Accessible** - Semantic HTML with ARIA labels
- 🎯 **Product Showcase** - 5 product sections with alternating layouts
- 📧 **Email-Ready** - Inline CSS for email client compatibility
- 🎭 **Reduced Motion** - Respects `prefers-reduced-motion` settings

## 🎨 Design Tokens

The template uses CSS variables for easy customization:

```css
--cyan: #2ec4e8
--cyan-dark: #18a8d0
--black: #1a1a1a
--grey-bg: #f4f4f4
--grey-text: #555555
--white: #ffffff
--border: #d9f0fb
```

## 🚀 Usage

1. Download the HTML file
2. Add all image files to an `images/` folder in the same directory
3. Open in a browser or email client to preview
4. Customize links and content as needed
